## _Second Homework of Android Workshop_

On this Application you can explore the entire world's map and bookmark whereever you want to visit. This application will save it for you and you can see them on the map later.

## Features

- Gives you the power to see the whole world's map
- Long press on anywhere to select the place
- Saves the bookmarks on SQLite so you can see them even if you quit the app
- **Dark mode!**
- Delete all your bookmars with just a tap
